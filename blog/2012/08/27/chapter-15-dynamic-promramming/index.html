    Chapter 15:Dynamic Promramming

    动态规划的4个步骤:

    1. 找到最优子结构
    2. 递归定义最优解
    3. 按自底向上的方式计算最优解的值
    4. 由计算结果构造一个最优解

    15.1-2 可以用归纳法

    15.1-4 的值只与相关,所以可以从$2n$缩减为$2$个._

    15.2-2

    MATRIX-CHAIN-MULTIPLY(A, s, i, j)
    if i < j
        x = MATRIX-CHAIN-MULTIPLY(A, s, s(i,j), j)
        y = MATRIX-CHAIN-MULTIPLY(A, s, s(i,j)+1, j)
        return MATRIX-MULTIPLY(x, y)
    else
        return Ai
